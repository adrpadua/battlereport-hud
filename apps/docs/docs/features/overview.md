# Features Overview

Battle Report HUD provides several integrated tools for enhancing the Warhammer 40,000 battle report viewing experience.

## Transcript Parser

The transcript parser (`packages/transcript-parser/`) processes battle report transcripts to extract:

- Unit names and factions
- Stratagems and abilities used
- Dice rolls and combat results
- Game state and scoring

## Chrome Extension

The Chrome extension (`apps/extension/`) enhances YouTube battle report videos with:

- Real-time unit identification
- Rules lookups during playback
- Overlay information on video

## Web Application

The web application (`apps/web/`) provides:

- Interactive battle report viewing
- Search and filtering capabilities
- Data visualization for game statistics

## MCP Server

The MCP (Machine Context Protocol) server (`mcp-server/`) offers:

- Warhammer 40K rules lookup via AI assistants
- Faction and unit data queries
- Stratagem and ability information
